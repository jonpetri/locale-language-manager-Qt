33 /**
34  * @class LlmQtLocaleLanguageManager LlmQtLocaleLanguageManager.h
35  * LlmQtLocaleLanguageManager is an utilitarian class that simplifies language choice management in a multilingual Qt application.
36  * It provide methods to deal with .qm files, app's active language, system language, language switch, choice storing in user settings...
37  * It handle all the possible error during .qm files analysis.
38  * Dialog's straight translation at running time is possible.
